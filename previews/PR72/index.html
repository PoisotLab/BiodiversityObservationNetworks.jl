BiodiversityObservationNetworks.jl ​

The purpose of this package is to provide a high-level, extensible, modular interface to the selection of sampling point for biodiversity processes in space. It is based around a collection of types representing point selection algorithms, used to select the most informative sampling points based on raster data.

This package is in development

The BiodiversityObservationNetworks.jl package is currently under development. The API is not expected to change a lot, but it may change in order to facilitate the integration of new features.

BONSampler

An abstract type that is the supertype for all methods for sampling BiodiversityObservationNetworks.

BiodiversityObservationNetworks.BiodiversityObservationNetwork Type
julia
BiodiversityObservationNetwork

A set of Nodes that together create a sampling design for monitoring biodiversity.

BiodiversityObservationNetwork extends the GeoInterface MultiPointTrait type.

source

BiodiversityObservationNetworks.Grid Type
julia
Grid

Grid is a type of BONSampler for generating BiodiversityObservationNetworks with Nodes structured as a systematic grid over the study area.

Arguments:

  • longitude_spacing

  • latitude_spacing

source

BiodiversityObservationNetworks.KMeans Type
julia
KMeans

KMeans is a BONSampler that generates BiodiversityObservationNetworks that aim to be representative of the distribution of environmental variables across an extent represented as a RasterStack.

Arguments:

  • k: the number of sampling sites

source

BiodiversityObservationNetworks.Raster Type
julia
Raster

A Raster stores gridded data. The internal representation of this data can be an SDMLayer (from SimpleSDMLayers.jl) or a Matrix.

Raster extends the RasterTrait type from GeoInterface.

source

BiodiversityObservationNetworks.RasterStack Type

A RasterStack is a set of Rasters, which all have the same resolution and extent, for storing multiple raster-like environmental variables

source

SimpleRandom is a BONSampler for sampling BiodiversityObservationNetworks where each location in the spatial extent has the same probability of inclusion.
