Simple Random Sampling ​

BiodiversityObservationNetworks.SimpleRandom Type
julia
SimpleRandom

SimpleRandom is a BONSampler for sampling BiodiversityObservationNetworks where each location in the spatial extent has the same probability of inclusion.

source

Example ​

Using a polygon ​

First, load the packages we will use for this example

julia
using BiodiversityObservationNetworks
using CairoMakie
using GeoMakie
import SpeciesDistributionToolkit as SDT

now sample a BiodiversityObservationNetwork

julia
num_nodes = 50
corsica = SDT.gadm("FRA", "Corse")
bon = sample(SimpleRandom(num_nodes), corsica)
BiodiversityObservationNetwork with 50 nodes

and plot

julia
f = Figure(size=(500, 500))
bonplot(f[1,1], bon, corsica, axistype=GeoAxis)
f

Now sample a BON. For a vector of polygons, SimpleRandom will sample from each polygon separately. Here we'll sample 10 nodes in each administrative region.

bon = sample(SimpleRandom(10), france_states)

Using a raster ​

First, load mean annual temperature for France using SDT

julia
france = SDT.gadm("FRA")
temp = SDT.SDMLayer(SDT.RasterData(SDT.WorldClim2, SDT.BioClim); layer=1, SDT.boundingbox(france)...)
temp = SDT.mask!(temp, france)
SDM Layer with 2310 Float32 cells
	Proj string: +proj=longlat +datum=WGS84 +no_defs
	Grid size: (59, 89)

now sample a BON

julia
bon = sample(SimpleRandom(50), temp)
BiodiversityObservationNetwork with 50 nodes

and plot

julia
f = Figure(size=(500, 500))
bonplot(f[1,1], bon, temp, axistype=GeoAxis)
f

Using a Raster Stack ​

Sampling with a RasterStack works nearly identically to using a Raster. First let's load a raster stack with two layers.

julia
france = SDT.gadm("FRA")
layers = RasterStack(SDT.SDMLayer[SDT.mask!(SDT.SDMLayer(SDT.RasterData(SDT.WorldClim2, SDT.BioClim); layer=i, SDT.boundingbox(france)...), france) for i in [1,12]])
RasterStack with 2 layers

now sample a BON

julia
bon = sample(SimpleRandom(50), layers)
BiodiversityObservationNetwork with 50 nodes

and plot

julia
f = Figure(size=(500, 500))
bonplot(f[1,1], bon, temp, axistype=GeoAxis)
f

Using a BON ​

SimpleRandom also works with BiodiversityObservationNetwork as a domain.

Lets generate 250 points using the Corsica Polygon first, and plot for good measure

julia
bon_candidate = sample(SimpleRandom(250), corsica)

f = Figure(size=(500, 500))
bonplot(f[1,1], bon_candidate, corsica, axistype=GeoAxis)
f

and then sample a subset of 100 of them

julia
bon = sample(SimpleRandom(100), bon_candidate)
BiodiversityObservationNetwork with 100 nodes

and plot

julia
f = Figure(size=(500, 500))
bonplot(f[1,1], bon, corsica, axistype=GeoAxis)
